# Mars-Rover-6wd-stair-climbing-robot
6WD stair-climbing Mars Rover with HC-12 telemetry and custom motor control algorithms.
## 3D Model Reference

Base chassis design sourced from:

Stair Climbing Rover – Printables  
https://www.printables.com/model/194299-stair-climbing-rover

Modifications in this project:
- Custom motor mounting
- Electronics compartment integration
- Wireless telemetry system (HC-12)
- Embedded motor control firmware
